WooCommerce Advanced Easy Shipping Plugin helps make Shipping process easy and convenient for E-commerce Store owners. The plugin makes it easier by offering different options to decide shipping rates based on different criteria.

== Description ==

The plugin allows you to make different shipping rate based on certain conditions. I.e., Cart, Location, Cart Specific. In addition to this, you can set extra shipping price by selecting specific parameters.

Advanced Easy Shipping For WooCommerce is a valuable tool for store owners for creating and managing complex shipping rules in their store without the help of a developer!

= Updating =

Automatic updates should work smoothly, but we still recommend you back up your site.

If you encounter issues with the shop/category pages after an update, flush the permalinks by going to WordPress > Settings > Permalinks and hitting “Save.” That should return things to normal.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `advanced-easy-shipping-for-wc.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
