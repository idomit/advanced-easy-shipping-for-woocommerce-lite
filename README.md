# Advanced Easy Shipping For WooCommerce Lite
The plugin allows you to make different shipping rate based on certain conditions. I.e., Cart, Location, Cart Specific. In addition to this, you can set extra shipping price by selecting specific parameters.

Advanced Easy Shipping For WooCommerce is a valuable tool for store owners for creating and managing complex shipping rules in their store without the help of a developer!

If you are not a developer, you can use the Advanced Easy Shipping For WooCommerce Lite [plugin page on WordPress.org](https://wordpress.org/plugins/advanced-easy-shipping-for-wc-lite/).

## PRO VERSION:
* Set up Different Conditions based on Geographical area/ User role
* Use of different operators to set up specific conditions
* Combination of two or more conditions
* Use of Either Conditions to add shipping
* shipping visible in the Cart, Checkout, Thank You page, and Order email
* Particular titles for particular shipping so that shipping can be differentiated easily
* Easy to add Tooltip for specific shipping
* Cart Based shipping options like on Subtotal / Subtotal ex. taxes / Tax / Quantity / Contains product / Coupon / Weight / Contains shipping class
* User Details Based shipping options like on Zipcode / City / State / Country / User role
* Product Based shipping options like on Width / Height / Length / Stock / Stock status / Category
